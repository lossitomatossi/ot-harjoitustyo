## Testaus

Testit suoritetaan komennolla
```
mvn test
```
Testikattavuusraportti luodaan komennolla
```
mvn jacoco:report
```
Kattavuusrapottia voi tarkastella avaamalla selaimella tiedosto target/site/jacoco/index.html
